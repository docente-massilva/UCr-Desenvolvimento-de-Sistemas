# Projeto Integrador

Sistema para gerenciamento da adoção de pets 🐾.

## Como executar

1. **Instale o Django:** `pip install -r requirements.txt`
2. **Execute o Serviço:** `python manage.py runserver`
3. **Acesse esse link:** Entre no link: http://127.0.0.1:8000

## Ativa o django admin

Garanta que o `django.contrib.admin` está em INSTALLED_APPS no arquivo settings.py

para criar o usuário Admin:

```
python manage.py createsuperuser
```
